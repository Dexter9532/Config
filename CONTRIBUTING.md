# Contributing

Thanks for contributing.

## What To Report

Use issues/PRs mainly for:

- Buggy plugin behavior
- Unexpected overrides/conflicts between plugins
- Keymaps/features not working as expected
- Regressions after plugin updates

## Bug Report Checklist

Please include:

- What you expected
- What happened instead
- Exact file(s) changed (if any)
- Steps to reproduce
- Neovim version (`nvim --version`)
- Lazy status/sync state (`:Lazy`)
- Relevant logs/errors/messages

## Pull Requests

1. Keep the fix focused.
2. Explain root cause.
3. Explain why the fix works.
4. Add quick test steps.

Small, isolated PRs are preferred.
